# Node.js

## Node.js 介绍

### Node.js 是什么

- 不是一门语言
- 不是一个库，框架
- 是一个 JavaScript 运行时环境
- 脱离浏览器运行 JavaScript代码

- 浏览器中的 JavaScript
  - EcmaScript
  - BOM
  - DOM
- Node.js 中的 JavaScript
  - 没有 BOM, DOM
  - EcmaScript
  - 提供了一些服务器级别操作的 API
    - 文件读写
    - 网络服务的构建
    - 网络通信
    - http 服务器
    - ......
- 构建于 Chrome V8 的引擎之上
- 事件驱动
- NIO API
- 轻量级高效

- npm node package manager ex : npm install jquery
